# My Health Diary

A kawaii-style digital health diary UI built with React and CSS, matching the provided design mockup.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).


## Structure

- `src/App.jsx` — layout shell
- `src/components/` — Sidebar, pages, health cards, journal, workouts, sticker panel, toolbar
- Connected apps use placeholders; wire up real integrations later
